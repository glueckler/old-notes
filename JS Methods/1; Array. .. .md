#Array. ..
----------




### .from()
```javascript
//The Array.from() method creates a new Array instance from an array-like or iterable object.

//ex
const bar = ["a", "b", "c"];
Array.from(bar);
// ["a", "b", "c"]

Array.from('foo');
// ["f", "o", "o"]

//ex
const baz = Array.from(foo, bar);

```
